\[ `HELP v\.2\.0` *\:* `2022\.07\.09` \]

/help — show this very message\.

_*Any text message*_ — bot will ignore it, unless there's a *tiktok*, *twitter* or *instagram* link in it, that can be parsed\.

If you send _*several*_ links at once in one message, *all links will be* ||_hopefully_|| *parsed*\.

\[ `TOGGLES` \]

/tiktok\_hd \(_default_\: *off*\) — toggle this on if you want to get video in HD quality \(may result in large videofile\)\.

/twitter\_hd \(_default_\: *off*\) — toggle this on if you want to get twitter photos as documents\.

/instagram\_hd \(_default_\: *off*\) — toggle this on if you want to get instagram photos as documents\.

/include\_link \(_default_\: *off*\) — toggle this on if you want to use source link as a caption to messages\.

\[ `OTHER COMMANDS` \]

/twitter\_style — cycle twitter post styles\. Only *3 styles* are available\.
